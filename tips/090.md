Two Markdown tricks to "nest" code blocks, i.e., to use backtick-based
code block syntax within a code block:

① More backticks for the outer block:

    ````` markdown
    ``` r
    print('Hello!')
    ```
    `````

② Tildes:

    ~~~ markdown
    ``` r
    print('Bonjour!')
    ```
    ~~~
